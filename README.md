## Run Locally

Clone the project:

```bash
git clone https://github.com/pervez-ahmed-shaik/pervez-ahmed-portfolio.git
```

Go to project directory:

```bash
cd pervez-ahmed-portfolio
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
